# 大学生活动招募平台需求报告

## 1. 项目概述

 **项目名称** ：大学生活动招募平台
 **目标用户** ：在校大学生
 **核心定位** ：面向校园的活动发布、发现与参与平台，整合社团活动、学校主办活动等校园资源

## 2. 核心功能模块

### 2.1 用户认证系统

#### 注册功能

* **必填字段** ：
* 手机号
* 邮箱
* 用户名
* 密码
* 再次确认密码
* 性别
* 年级
* 学院
* 专业
* **可选字段** ：
* 爱好（用于个性化推荐）

#### 登录功能

* 支持手机号/邮箱登录
* 密码验证
* 忘记密码流程：
  * 身份验证（手机号+邮箱双重验证）
  * 验证通过：允许修改密码
  * 验证失败：提示身份验证失败

### 2.2 用户信息管理

* 基础信息存储
* 我的报名记录
* 我的收藏列表
* 历史浏览记录（可选）

### 2.3 活动管理系统

#### 活动数据结构

* 活动ID（唯一标识）
* 活动名称
* 发布人信息
* 活动地点
* 活动时间
* 活动简介（包括收获：志愿时、综测加分等）
* 封面图片
* 面向人群（专业/年级筛选）
* 招募人数
* 活动标签（分类：文娱类、就业类等）

## 3. 界面功能需求

### 3.1 注册登录界面

* 表单式设计
* 响应式布局
* 输入验证

### 3.2 主界面功能

1. **搜索功能**
   * 关键字搜索
   * 高级筛选：
     * 按专业筛选
     * 按时间筛选
     * 按活动类型筛选（学院办/个人办/校级）
2. **活动报名**
   * 报名确认弹窗
   * 用户信息传输至活动发布方
3. **活动发布**
   * 完整的活动信息填写表单
4. **推荐系统**
   * 个性化推荐算法（可选）
5. **排行榜**
   * 热度排行榜
   * 最新发布列表
6. **个人中心**
   * 个人信息查看
   * 收藏管理
   * 参与历史
7. **AI客服助手**
   * 智能客服支持

## 4. 技术接口规范

### 4.1 核心API接口

#### 获取活动列表接口

**text**

复制

下载

```
GET /api/activities/
```

 **查询参数** ：

* keyword: 搜索关键词
* benefits: 收获类型筛选
* audience: 面向人群筛选
* categories: 活动分类筛选
* time_range: 时间范围筛选
* sort_by: 排序方式
* page: 页码
* page_size: 每页数量

#### 活动报名接口

**text**

复制

下载

```
POST /api/activities/{activityId}/join
```

#### 报名状态检查接口

**text**

复制

下载

```
GET /api/activities/{activityId}/join-status
```

#### 浏览量统计接口

**text**

复制

下载

```
POST /api/activities/{activityId}/view
```

### 4.2 数据验证规则

* 活动时间：有效ISO格式
* 招募人数：必须大于0
* 当前参与人数：0到最大人数之间
* benefits和target_audience：必须为数组格式

### 4.3 错误处理规范

**json**

复制

下载

```
{
  "success": false,
  "message": "错误描述",
  "status": 错误码
}
```
